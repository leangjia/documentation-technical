使用KeyCDN

本文将指导您完成一个KeyCDN_帐户与您Odoo供电网站的设置。

步骤1：创建在KeyCDN仪表板拉区

创建安全区时，可启用的CORS选项:guilabel:[UNKNOWN NODE problematic]先进features`子菜单。 （稍后更多）

一旦这样做，你得稍等一下，而KeyCDN_爬行你的网站。

注解

一个新的URL已产生针对你的区域，在这种情况下，它是``的http：// pulltest-b49.kxcdn.com``

步骤2：用您的区域配置odoo实例

在Odoo后端，转到:guilabel:[UNKNOWN NODE title_reference]：菜单，然后激活CDN支持，并复制到/粘贴区网址:guilabel:[UNKNOWN NODE problematic]CDN基础URL`字段。

现在，您的网站使用CDN匹配的资源:guilabel:[UNKNOWN NODE problematic]CDN filters`正则表达式。

你可以看看你的网站的HTML，以检查CDN集成正常工作。

为什么要激活CORS？

在某些浏览器（Firefox和Chrome在写本新闻时）安全限制阻止远程链接CSS文件，以获取对同一外部服务器资源相对。

如果不激活的CDN区CORS选项，在默认Odoo网站比较明显的随之而来的问题将是缺少字体真棒图标，因为字体真棒CSS声明的字体文件就不会被加载上远程服务器。
